# Curtin ML Group — Handbook

Welcome. This is the one place that explains **how we work** — the tools, the workflow, and what's expected. If you're new, start with [Getting started](getting-started.md).

We do machine learning and AI for structural engineering, across a range of projects. Current projects include:

| Project | Focus |
|---|---|
| **DoT** | Vehicle-barrier collision simulation |
| **Linkage** | Infrastructural Digital Twin |
| **DECRA** | Agentic AI for SHM |

…and more. Each of us works on a specific **workstream** within a project — this handbook is how those pieces stay coordinated.

## The idea in one sentence
Everything about your work lives on GitHub — code, tasks, experiments, decisions — so it's visible, reproducible, and easy to pick up. Large files (datasets, 3D, checkpoints) live in the right specialist tool, with a pointer from GitHub.
